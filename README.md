# pyXLMS Manuscript

Results, figures, and code for the manuscript of [pyXLMS](https://github.com/hgb-bin-proteomics/pyXLMS).

The structure of this repository is as follows:

- `export`: Exported files from pyXLMS.
- `fasta`: Used FASTA file.
- `figures`: Figure files for the manuscript.
- `ms`: Details about the used mass spectrometry data.
- `params`: Details about the used parameters for crosslink search.
- `plots`: Plots created with pyXLMS.
- `results`: Results from the crosslink search engines, AlphaFold2 and AlphaLink2.
